# Marco Labs

This project is my [personal site](https://marco.deno.dev).

## Tech Stack

- [Deno](https://deno.land/)
- [Fresh](https://fresh.deno.dev/)
- [Ansible Vault](https://docs.ansible.com/ansible/latest/vault_guide/)

### Install CLI

```sh
# local
curl -sSfL http://localhost:8000/install | bash
```

### Usage

Here we have some useful commands:

```sh
# format code
deno fmt

# start project
deno task start

# encrypt and decrypt .env
deno task decrypt
deno task encrypt
```
